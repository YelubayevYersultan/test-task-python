# Test Task Solution (Python)

This repository contains a solution to a basic programming test task.

## Task Description

The program performs the following:

1. If the entered number is greater than 7, it prints "Hello".
2. If the entered name is "John", it prints "Hello, John". Otherwise, it prints "There is no such name".
3. It accepts a numeric array as input and outputs the elements that are multiples of 3.
4. It also contains an analysis of a bracket sequence.
